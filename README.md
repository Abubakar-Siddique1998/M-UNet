# M-UNet
### Download ISPRS Datasets (Potsdam & Vaihingen) from [ISPRS](https://www.isprs.org/) 

### Download Weights from [Google Drive](https://drive.google.com/drive/folders/1oTcPRhWKT3XH7Y1CtsNnFr-0thkgGeEg?usp=share_link)

### Training
- Download the dataset to /root/data
- Run python train.py
### Testing
- Download the weigths from Google Drive and save to /root/saved_models
- Run python test.py
### Requirements 
- Python==3.9
- Pytorch==1.10
- Scikit-learn==1.0.2
- Opencv==4.6.0
- Tensorboard==2.10.1
